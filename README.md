# TDAs - Algoritmos 2

Proyecto de práctica para la materia Algoritmos 2.  
Contiene especificaciones e implementaciones de distintos Tipos de Datos Abstractos en Java.

## Contenido

El proyecto incluye los siguientes TDAs:

- Pila
- Cola
- Cola con prioridad
- Conjunto
- Diccionario simple
- Diccionario múltiple

## Estructura del proyecto

```text
especificaciones/     Interfaces de los TDAs
implEstaticas/        Implementaciones con estructuras estáticas
implDinamicas/        Implementaciones con nodos dinámicos
ejerciciosEstaticos/  Ejercicios y operaciones usando TDAs estáticos
ejerciciosDinamicos/  Ejercicios y operaciones usando TDAs dinámicos
tests/                Clases de prueba con métodos main
